Cruise  SS 02/98

 Data processing completed by

Principal Investigator Dr. Steve Rintoul, CSIRO Division of Marine Research, Hobart, Tasmania.

General

Following are details of the hydrology processing done for S9802. See Table 1 for specific edits.

In general, the data seems to be of good quality considering the number of stations completed (101). Stations 37 and 81 were fired out of order, but the data was sorted out and has been corrected. Station 9 also appears to have been fired out of order, although no notes were made on the log sheets at the time.

Nutrients

A considerable amount of time was spent processing the nutrient data, mostly because DAPA was used to collect the data. External standards were used during all runs and indicate that overall there was some determinate error associated with the nitrate and phosphate results, but not with the silicate results. Accuracy and precision data for each run is given in Table 2. The results for the entire voyage show errors of -3.99% for nitrate+nitrite at 10.00uM, 1.95% for silicate at 10.00uM and -3.79% for phosphate at 1.00uM. Voyage precision (calculated as the coefficient of variation for the pooled standard deviations) was 0.37% for nitrate +nitrite, 1.02% for silicate and 0.45% for phosphate. Detection limits were not determined for this voyage, but the following can be used as a guide: 0.07uM nitrate+nitrite, 0.10uM silicate and 0.013uM phosphate. Values in the data set that are reported below these concentrations should be treated with caution.

For specific edits to the sample data, see Table 3.

Salinities

Edits to salinity data are shown in Table 4. There are a few samples where the Hydro salinity differed from the CTD value markedly. Some of these were deleted. Figure 1 shows a plot of CTD salinity - Hydro salinity for the entire voyage.

Temperature

Much unprotected thermometer data was deleted due to large discrepancies with the CTD temperature data. Edits are included in Table 5.

Dissolved Oxygen

The CTD dissolved oxygen results were not calibrated during the CTD processing. There was no CTD downcast data to compare the hydro results to. Hydro dissolved oxygen results were plotted against depth and some data points were deleted due to incorrect data entry on the voyage. Most of the data was acceptable. Edits are listed in Table 6.

Archiving

Data processing was completed on the 26/10/98 and archived on the 27/10/98. Problems were encountered with archiving the data with respect to niskin bottle numbers, as those listed on the CTD sheets are 4 digit numbers. These numbers were shortened by removing the second digit in the number, which in all cases was a one. For example, bottle number 1709 became 709 (signifying a 1.7-litre bottle, number 09).
